# LuminaDB Frontend

This folder contains the modern user interface for LuminaDB. It is built using **React** (bootstrapped with Vite) and styled with **Tailwind CSS**.

## Features
- **Glassmorphism UI**: Uses beautiful blur effects and a light-themed, modern palette (`#F8FAFC` background, crisp white cards, and Soft Blue `#3B82F6` accents).
- **Responsive Layout**: Designed to work on desktops and mobile automatically.
- **Real-Time Previews**: Dynamically renders the exact raw MongoDB Query Language (MQL) JSON that was generated by the backend so users have full transparency into the AI translation process.

## Getting Started

1. Navigate to this directory:
   ```bash
   cd LuminaDB/frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
   
The app will automatically be available at `http://localhost:5173`.

## Architecture Details
- **`Dashboard.jsx`**: The main page wrapper holding the background gradients and UI logic.
- **`SearchBar.jsx`**: The glassmorphic input field that captures the natural english prompt.
- **`ResultsTable.jsx`**: Dynamically maps any JSON payload returned by MongoDB into a paginated data table.
